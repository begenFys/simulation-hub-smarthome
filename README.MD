## Настройка окружения

1. Установить зависимости из Pipfile(заведомо установить pipenv)
```
pipenv sync
```

2. Запустить виртуальное окружение
```
pipenv shell
```

## Запуск проекта
1. Запустить main.py файл, передав **url сервера** и **адрес хаба в hex виде**
```
python main.py <url> <address>
```
